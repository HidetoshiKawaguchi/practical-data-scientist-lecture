# Wine-quality プロジェクト
これは、ワイン品質推定プロジェクトにおけるEDA・実験を管理するリポジトリで

## リポジトリの構成
一回分の実験とそれに関連するEDAを一つのディレクトリで管理しています。具体的には、以下のような規則でIDを振ることで管理します。

- `exp001`
- `exp002`
- ...

ぞれぞれのディレクトリで独立して実験を管理します。以下のルールを遵守すること。

- 各ディレクトリの README.md に以下の内容を含めること
  - 実験の背景
  - 自身が実施した作業や分析の概要
  - 実験の再現方法
      - データのダウンロード方法
      - 環境構築方法 (仮想環境等)
      - 実験コードやスクリプトの実行方法
  - 実験を通じて得た知見
- 一度実行が完了したEDA・実験のコードは変更しない（変更する場合は実験結果のデータ等は破棄すること）
