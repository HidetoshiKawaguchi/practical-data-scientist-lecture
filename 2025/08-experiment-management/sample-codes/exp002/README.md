# Wine-quality プロジェクト 実験002

Wine-qualityプロジェクトに関する実験を行う。
実験001からの継続実験にあたり、多層パーセプトロンではなくSVR (Support Vector Regression)を適用して効果を確かめる。

## 主なファイルの説明
各ファイル、フォルダの概要は以下の通りです。なお、実験対象のデータは実験001のものを参照するため、本フォルダには同梱されていない。

- `exp002.py`: 予測モデルの構築と評価用のエントリーポイントとなるコード（実験用コード）
- `svr_exp.py`: 1回分の実験を行う関数を含んだコード
- `logs`: 実験のログファイルを保存する。`level_6.py`が実行されると作成される
- `data`: EDAと実験で使うデータや、結果のファイルを保存する用のディレクトリ
    - `results`: 実験結果のJSONファイルが保存されているディレクトリ。`level_6.py`が実行されると作成される

## 環境構築
各プログラムはPythonで動きます。仮想環境の管理にはpoetry を使っている想定です。
以下のコマンドで環境構築ができます。

- `poetry install --no-root`: パッケージのインストール
- `poetry shell`: 仮想環境の適用

以降の実行は、すべて上述したコマンドを実行しているという前提とします。

### データのダウンロードについて
本実験では、前段の実験001のデータを参照します。

### 実験の再現方法
実験は、この`README.md`のあるフォルダをカレントディレクトリとして、以下のコマンドで実行できます。

```
PYTHONPATH=.. python exp002.py
```

実行の標準出力例
```
Grid(SVR): 100%|██████████████████████████████████████████████████████████████| 12/12 [00:04<00:00,  2.48it/s]
```

実行後、以下のコマンドで結果ファイルを集計したCSVを作成できます。

```
python summarize_results.py --input data/results --output data/results_summary_svr.csv 
```

実行の標準出力例
```
Wrote summary CSV to: /<>絶対パス/exp002/data/results_summary_svr.csv
```
